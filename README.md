# Student Freelancer Marketplace

A simple full-stack authentication system for a student freelancer marketplace built using PHP, MySQL, Bootstrap, HTML, and CSS.

## Features
- User Registration
- Secure Login System
- Password Hashing
- Session Authentication
- Protected Dashboard
- Logout Functionality
- Responsive UI using Bootstrap

## Technologies Used
- PHP
- MySQL
- Bootstrap 5
- HTML5
- CSS3

## Project Purpose
This project was created to practice backend web development concepts such as authentication, database connectivity, session handling, and user management using PHP and MySQL.
